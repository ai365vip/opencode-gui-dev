# Claudix - Claude AI 编程助手

![VS Code Extension](https://img.shields.io/badge/VS%20Code-Extension-blue?logo=visual-studio-code)
![Built with TypeScript](https://img.shields.io/badge/Built%20with-TypeScript-blue?logo=typescript)
![License](https://img.shields.io/badge/License-AGPL--3.0-blue)

将Claude AI直接集成到VSCode中的智能编程助手。

## 主要功能

### 💬 智能对话编程
- 与Claude AI进行自然语言对话
- 实时流式响应，无等待感
- 支持代码解释、优化建议、Bug修复

### 📁 文件操作
- 智能读取和编辑文件
- 批量文件分析和修改
- 支持各种编程语言的语法高亮

### 🔧 代码分析工具
- 代码结构分析和重构建议
- 性能优化提示
- 最佳实践指导

### 🗂️ 会话管理
- 保存对话历史记录
- 快速切换不同项目会话
- 支持会话恢复和继续

### ⚡ 终端集成
- 执行命令行操作
- 自动化脚本运行
- 实时查看执行结果

### 🛡️ 权限控制
- 安全的工具访问机制
- 用户确认重要操作
- 透明的权限提示

### 🤖 多模型支持
- 支持不同的Claude模型
- 自定义模型配置
- 灵活的模型切换

## 快速开始

### 1. 打开Claudix
- 点击VSCode左侧活动栏的Claudix图标
- 或使用快捷键打开侧边栏

### 2. 开始对话
- 在输入框中输入问题或需求
- 支持自然语言描述编程任务
- 例如："帮我优化这个函数的性能"

### 3. 文件操作
- Claudix可以自动读取和修改项目文件
- 批量处理多个文件
- 操作前会征求你的确认

### 4. 会话管理
- 所有对话会自动保存
- 可以随时切换到历史会话
- 支持不同项目的独立会话

### 5. 权限确认
- 重要操作会弹出确认对话框
- 仔细查看要执行的操作
- 点击"允许"或"拒绝"控制权限

- Node.js >= 18.0.0
